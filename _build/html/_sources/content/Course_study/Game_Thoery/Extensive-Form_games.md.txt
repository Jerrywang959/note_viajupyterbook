# 扩展博弈和逆向归纳

## Game Trees

### sequential rationality

序贯理性：假设每个人在它的选择时都采取的理性的选择。后面作出决策的人知道前面人的选择，前面作出决策的人知道自己的决策会被后面看到。

## 序贯博弈

序惯博弈的要素

1. 博弈的玩家集合
2. 玩家的收益函数
3. 做决策的顺序
4. 每一个玩家可以做博弈时的选择
5. 玩家做决策时拥有的信息
6. 外生事件的概率分布
7. 以1-6为代表的变形式博弈的结构是所有博弈者的共识
