myfriends = ["Ted", "Robyn", "Barney", "Lily", "Marshall"]

    friend = myfriends[i]

If条件语句¶

if *condition 1*
    *option 1*
elseif *condition 2*
    *option 2*
else
    *option 3*
end

举例¶

julia> N =4
4

julia> if (N % 3 == 0) && (N % 5 == 0) # `&&` means "AND"; % computes the remainder after division
           println("FizzBuzz")
       elseif N % 3 == 0
           println("Fizz")
       elseif N % 5 == 0
           println("Buzz")
       else
           println(N)
       end
4

if语句的其他形式¶

三元运算符¶

a ? b : c

这等同于

if a
    b
else
    c
end

使用语句连接符¶

julia> x=-4
